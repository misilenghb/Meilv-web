添加以下必需密钥：
```

NEXT_PUBLIC_SUPABASE_ANON_KEY
SUPABASE_SERVICE_ROLE_KEY
NEXT_PUBLIC_SUPABASE_STORAGE_URL
SUPABASE_STORAGE_KEY_ID
SUPABASE_STORAGE_ACCESS_KEY
```
